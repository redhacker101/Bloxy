Game Upload Requirements

Every uploaded game must follow the Alpha rules before it can be approved.

Linux Build Required

Your game must run on Bloxy's Linux servers using Godot headless mode if they are multiplayer.

Server Pack Required

You must upload a working server pack that can run on a Bloxy server.

Client Pack Required

You must upload a client pack that players can launch through the Bloxy app.

Bloxy Auth Required

Your game must support Bloxy launch tokens so users cannot fake usernames or accounts.This will be provided in the SDK.

Must Be Lightweight

If your game does not run well on low-end devices, it may be rejected.
