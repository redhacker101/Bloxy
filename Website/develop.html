Game Upload Requirements

Linux Build Required

Your game must run on Bloxy's Linux servers using Godot headless mode.

Multiplayer Support

Single-player-only games are not accepted during Alpha. Your game must support real multiplayer.

Server Pack Required

You must upload a working server pack that can run on a Bloxy server.

Client Pack Required

You must upload a client pack that players can launch through the Bloxy app.

Bloxy Auth Required

Your game must support Bloxy launch tokens so users cannot fake usernames or accounts.

Must Be Light Weight

If your game dosent run well on potatos it will be rejected.
